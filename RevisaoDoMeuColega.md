# Avaliação do Sistema

## 1. O sistema está funcionando com as funcionalidades solicitadas?

**Resposta:**

A criação da prova não oferece a opção de selecionar as questões. Só é possível selecionar ou modificar as questões ao editar a prova posteriormente. Durante a criação, o sistema inclui automaticamente todas as questões existentes.

Além disso:
- O gabarito é gerado com os números das provas, porém não inclui as alternativas corretas.
- Não é possível anexar o arquivo CSV do gabarito.
- Também não é possível adicionar o CSV com as respostas dos alunos.
- Existem campos para digitação manual dessas informações, mas não foi possível fazê-los funcionar.

As demais funcionalidades estão operando normalmente.
